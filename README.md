# Bulty-Dolui-AWS-Project
Scalable Airline Data Processing using AWS
A cloud-based big data project that analyzes and processes the Airlines Delay Dataset to identify flight delay patterns and performance insights. The project leverages AWS services for scalable data storage, processing, and integration with GitHub for version control and automation.

Key Features:
Used Amazon S3 for large-scale dataset storage.
Processed and analyzed flight delay data using AWS Glue / Athena / EMR.
Built scalable ETL pipelines to clean, transform, and query millions of records.
Integrated with GitHub for CI/CD and collaborative workflow management.
Generated insights on flight delays, carriers, airports, and seasonal trends.

Tech Stack: AWS (S3, Athena, EMR) · Python · SQL · GitHub · Pandas · PySpark
